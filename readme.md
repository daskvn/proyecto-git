# Proyecto Git

Este proyecto simula un flujo de trabajo colaborativo con Git y GitHub.

## Funcionalidades
- **Navbar**: agregado en la rama `feature/navbar` y fusionado en `main`.
- **Footer**: agregado en la rama `feature/footer` mediante Pull Request.

## Flujo de trabajo realizado
1. Creación de repositorio local con `git init`.
2. Primer commit con `index.html` y `README.md`.
3. Subida del proyecto a GitHub (`git push`).
4. Creación de ramas para nuevas funcionalidades (`feature/navbar`, `feature/footer`).
5. Commit y merge de la rama `feature/navbar`.
6. Pull Request y merge de la rama `feature/footer`.

## Ramas utilizadas
- `main`
- `feature/navbar`
- `feature/footer`
